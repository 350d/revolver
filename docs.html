Custom Options

Custom options are used to customize one instance of Revolver without it affecting another. They should be passed directly to the revolver plugin.

Option Type Default Description
autoPlay boolean true Set to false if you do not want Revolver to begin playing immediately.
rotationSpeed Integer 4000 Number of milliseconds to stay on each slide before transitioning to the next.
transition object - An object with options specific to transitions only.
speed Integer 1000 Number of milliseconds the transition should last.
type String "fade" The type of transition to use for all transitions. See here for the full list of transitions.
direction String "left" The direction the transition should take. Can be set to "up", "down", "left", or "right".
sliderClass string "slide" The class name that Revolver uses to determine the wrapper elements for each slide.

API Methods

These methods are how you will build your controls for the slider.

Method Description
first() The direction the transition should take. Can be set to "up", "down", "left", or "right".
goTo(slide) The class name that Revolver uses to determine the wrapper elements for each slide.
last() The class name that Revolver uses to determine the wrapper elements for each slide.
next() The class name that Revolver uses to determine the wrapper elements for each slide.
pause() Number of milliseconds to stay on each slide before transitioning to the next.
play() Set to false if you do not want Revolver to begin playing immediately.
previous() The class name that Revolver uses to determine the wrapper elements for each slide.
reset() Number of milliseconds the transition should last.
restart() The type of transition to use for all transitions.
stop() An object with options specific to transitions only.
