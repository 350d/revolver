Custom Options

You can change the default options by simply passing your own custom options as the first argument to the .revolver() plugin.

Here's a quick example with all options so you can copy and paste into your project: 

$('#my_slider').revolver({
    autoPlay:           true,
    rotationSpeed:      4000,
    transition: {
        direction:      'left',
        easing:         'swing',
        speed:          500,
        type:           'fade'
    },
    slideClass:         'slide'
});

Please note that this does not overwrite Revolver's defaults, the new options are simply merged with the defaults and stored in another property. You will need to declare any custom options for each instance of Revolver. For example, if you want to disable autoPlay for all of your sliders, you will need to do something like this: 

var my_options = { autoPlay: true };

$('#slider1').revolver(my_options);
$('#slider2').revolver(my_options);

autoPlay: true Boolean

Set to false if you do not want Revolver to begin playing immediately.

// turn off autoPlay
$('#my_slider').revolver({autoPlay: false});
Back to top

rotationSpeed: 4000 Integer

Number of milliseconds to stay on each slide before transitioning to the next.

// speed up the slider to 1 second per slide
$('#my_slider').revolver({rotationSpeed: 1000});
Back to top

transition Object

The transition option is just a namespace for the options that are specific to the transition itself.

Back to top

transition.direction: "left" String

The direction the transition should take. Can be set to "up", "down", "left", or "right".

// let's slide up
$('#my_slider').revolver({transition: {type: 'slide', direction: 'up'}});
Back to top

transition.easing: "swing" String

By default, there are only two easing methods available that's included in the jquery library: "swing" and "linear". To get more, go get the jquery easing plugin, include it in your document before Revolver, then replace "swing" with the name of the new easing method.

// let's slide up
$('#my_slider').revolver({transition: {type: 'slide', direction: 'up'}});
Back to top

transition.speed: 500 Integer

Number of milliseconds the transition (animation) should last.

// speed up the animation to 300 ms
$('#my_slider').revolver({transition: {speed: 300}});
Back to top

transition.type: "fade" String

The type of transition to use for all transitions. See here for the full list of transitions.

// use the "slide" transition
$('#my_slider').revolver({transition: {type: 'slide'}});
Back to top

sliderClass: "slide" String

Since Revolver doesn't assume every child element in the container is a slide, it needs to know which elements are, in fact, a slide. 

<div id="top_news">
    <h2 class="headline">How to make pure awesomeness with Revolver.js</h2>
    <h2 class="headline">How to take your javascript skills to the next level</h2>
    <h2 class="headline">Embrace prototypes, they're awesome!</h2>
</div>

// use a custom class for my slides
$('#top_news').revolver({sliderClass: 'headline');

Custom Events

Revolver's custom events allow you to "hook" into certains parts of the script and add your own functionality. You add a custom event handler the same way you declare custom options, just pass it to the revolver plugin.

Event Gets triggered...
onPause ...when the pause() api method is run.
onPlay ...when the play() api method is run.
onReady ...when Revolver is fully setup and ready to go.
onRestart ...when the restart() api method is run.
onStop ...when the stop() api method is run.
onTransitionStart ...when a transition begins to animate.
onTransitionComplete ...when a transition's animation has ended.
